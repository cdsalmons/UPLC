h2. Dependencies

* PHP >= 5.3
* Images needs ImageMagick or PHP GD

h2. What is it?

The UPLC is exactly what its name implies: a big collection of PHP libraries. At the moment, it includes the following libraries:

* Compression
* Files
* Images
* Input
* Output
* Shell

And there are plans of adding many more, including:

* Database
* ExtID (OpenID + OAuth)
* Git
* HTTP
* Preg (Regular Expressions)
* SourceMin (CSS/JS minification)
